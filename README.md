# 🏥 Healthcare Analytics Dashboard

## 📊 Project Overview

This project is an interactive Healthcare Analytics Dashboard built using Power BI. It provides insights into patient data, revenue trends, and hospital performance.

---

## 🚀 Key Features

* Total Patients Tracking 👨‍⚕️
* Revenue Analysis 💰
* Paid vs Unpaid Insights ✅
* Average Stay Duration 🏥
* Appointment Completion Rate 📈
* Department-wise Performance 📊
* Doctor Performance Analysis 👨‍⚕️

---

## 📈 Dashboard Insights

* Identified top-performing departments based on revenue
* Tracked patient trends over time
* Analyzed high-value patients
* Measured hospital efficiency using completion rate and stay duration

---

## 🛠️ Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Excel / CSV Data

---

## 📂 Project Structure

* `data/` → Raw dataset
* `dashboard/` → Power BI file (.pbix)
* `images/` → Dashboard screenshots
* `sql/` → Queries used for analysis
